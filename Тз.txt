1. Введение  
1.1. Цель проекта  
Разработка многофункционального Telegram-бота на Python с 7-10 командами, включая авторизацию/аутентификацию и справочную систему.  
---
2. Общее описание  
2.1. Назначение бота  
Бот предоставляет пользователям доступ к набору полезных функций через Telegram.  
---
3. Функциональные требования  
3.1. Обязательные функции  
1. Авторизация/аутентификация  
   - Регистрация по email/логину и паролю.  
   - Вход в аккаунт.  
   - Смена пароля.  
   - Выход из аккаунта.  
2. **Команда /start**  
   - Приветственное сообщение с кратким описанием возможностей бота.  
3. **Команда /help**  
   - Вывод списка всех доступных команд с их описанием. 

4. ** команда /add purchase sum category** 
-Добавление покупки в планировщик бюджета, категории расходов заданы заранее и выбираются из списка
5. ** команда /analyze** 
-выводит анализ по категориям расходов за последний месяц.
6. ** команда /salary ** 
-добавляет сумму к бюджету, возможно пополнение в разных валютах(тенге, рубли), автоматически пересчитывает в рубли
7. ** команда /ratio savings/consumption **  
-задает распределение сбережений/потребления, часть до 100 % идет в категория pleasure, которая предназначена для выполнения мечт
8. **команда /dream Name Sum** 
-ставит цель на конкретную мечту, если предыдущая мечта к этому моменту не достигнута, то она удаляется, накопленные деньги переходят на новую мечту.
9. **команда /show dream** 
-показывает процент достижения мечты
10. ** команда /pocket** 
-показывает текущий остаток на потребление, остатки с прошлого месяца переходят на следующий. 
3.2. Дополнительные функции (5-7 на выбор разработчика
Примеры возможных функций (разработчик может выбрать или предложить свои)
Управление задачами 
  - /addtask [описание] — добавить задачу.  
  - /tasks — показать список задач.  
  - /deltask [ID] — удалить задач
Конвертер валют
  - /convert [сумма] [из валюты] [в валюту] (использует API, например, exchangerate-api.com)
Прогноз погоды
  - /weather [город] — вывод текущей погоды (интеграция с OpenWeatherMap)
Личные заметки
  - /newnote [текст] — создать заметку.  
  - /notes — просмотреть все заметки
Голосования/опросы  
  - /poll [вопрос] [варианты] — создать опросы
Случайные советы  
  - /advice — вывод случайного совета из предустановленного списка
Поиск в Википедии  
  - /wiki [запрос] — краткая выдержка из стать
Напоминания
  - /remind [время] [текст] — установить напоминание.  

4. Нефункциональные требования
Язык и технологии: Python 3.10+, библиотека python-telegram-bot (как пример, можно другую)
